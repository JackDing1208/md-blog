# 函数

## 基本定义

一行或多行语句组成，完成特定的任务，相对独立代码片段

## JS里函数的特点

- 支持匿名（lamda）
- 私有的块级作用域（闭包）
- 可以指定this（call、bind）
- 可以创建对象（new语法糖）
- 参数相同，返回值可能不同(非函数式)

### 函数的环境在定义时确定

函数定义时只确定环境对应作用域内哪个变量，变量的值还是要看具体执行时候的情况

```javascript
let xxx = 1
function f1 (){
    console.log(xxx)
}
function f2 (){
    var xxx  = 2
    f1()
}
xxx = 3
f2() // 3
xxx = 4
```
